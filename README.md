1. Скачайте архив
2. Распакуйте в любом удобном месте
3. Скачайте Node.js
4. В cmd проверьте правильность установки:
node -v
npm -v
5. Если все верно, вы должны увидеть номера версий
6. В папке проекта выполните команды:
npm install - установка зависимостей
npm run asbuild - запуск сборки
npx serve . - быстрый старт сервера
7. Откройте браузер и введите http://localhost:3000
8. Выберите фото и сравните время выполнения JS и WASM
9. При необходимости перезагрузите страницу
