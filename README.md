1. Скачайте архив
2. Распакуйте в любом удобном месте
3. Скачайте Node.js
4. В cmd проверьте правильность установки:
```node -v```
```npm -v```
6. Если все верно, вы должны увидеть номера версий
7. В папке проекта выполните команды:
npm install - установка зависимостей
npm run asbuild - запуск сборки
npx serve . - быстрый старт сервера
8. Откройте браузер и введите http://localhost:3000
9. Выберите фото и сравните время выполнения JS и WASM
10. При необходимости перезагрузите страницу
